# Best-Movie APP

## 1. Spécification

- Possibilité de partager un film
- Utilisation de Firebase pour:
  - Authentification
  - Envoi de mail
  - Base de donnée
- Possibilité de se construire une collection personnalisé
- Gestion complète du Dark Mode

## 2. Installation

```
git clone https://github.com/enzotoyos/Best-Movie
npm install
expo start
```
